The Eye surfaced **[[evidence-learning-canon]]** as a gap aligned with the north-star (*become a
precision-ai-trainer*). The canon is well-written but **asserted** — this plan turns it into something tested.

## Open questions (new pupil nodes)

## Study directions
- **Ground each of the six moves in established literature** (Wave-2 web research, *sourced — never fabricated*):
  - *attack your own hypothesis* ↔ red-teaming / falsificationism;
  - *learn from non-conversion* ↔ hard-negative mining / contrastive learning / preference data;
  - *separate proof from usefulness* ↔ internal vs. external validity, eval design;
  - *require controls* ↔ ablation / one-variable experiments.
  Each confirmed mapping → a sourced `03_wiki/concepts/<c>.md` linked back to the canon.
- **Stress the spine.** Re-read [[structured-adversarial-evidence-learning]] and find the weakest link in
  `candidate → hypothesis → evidence → controlled test → reachability → impact → conversion`.

## Project seeds
- **Cross-domain transfer demo** — the smallest public-data instance of the full loop in a non-security field; the
  honest test of the transfer claim and a portfolio artifact ([[open-ideas-not-edge]]: publish the method, not the edge).
- **A blocker-label dataset spec** — a schema that makes non-conversion machine-usable; the bridge toward
  precision-ai-trainer and a system that [[growing-indefinitely|grows indefinitely]].
